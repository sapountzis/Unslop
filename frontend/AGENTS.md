# AGENTS.md – Frontend (Static Site)

You are working in the **static site** served at `https://getunslop.com`.

v0.1 scope:

- Host a minimal landing page and policy/support pages.
- Provide a stable place for:
  - Chrome Web Store install link
  - Privacy Policy
  - Support contact
  - (Optional) Terms

Out of scope:

- No login UI.
- No dashboard.
- No tracking pixels, no analytics scripts, no cookies banner complexity.
- No “app” functionality; this is purely informational.

Implementation guideline:

- Keep it as plain static HTML/CSS.
- No build step required.
- All URLs and emails should be placeholders or environment-independent constants.
