# Unslop Frontend (Rebrand v2)

Astro + Tailwind + React + Framer Motion single-page rebrand for getunslop.com.

## Commands

```bash
cd frontend
npm install
npm run dev
npm run build
```

## Routes

- `/` (5-slide scroll snap landing page)
- `/privacy.html`
- `/support.html`
- `/terms.html`

## Design Notes

- Background: `#F9F8F4`
- Headings: Newsreader serif
- UI/data text: Geist Mono
- No stock imagery
- CSS scroll snap for slide deck behavior
